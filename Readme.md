# 🌦️ Weather App

A simple and clean Weather App that allows users to search for any location and get real-time weather data using the WeatherAPI.

---

## 🚀 Features

* 📍 Fetch weather data based on user input location
* 🌡️ Displays temperature, condition, and other weather details
* 🔎 Search functionality for any city or place
* ⚡ Fast and responsive UI
* 🌐 Real-time data using WeatherAPI

---

## 🛠️ How It Works

1. User enters a location (city name)
2. The app sends a request to the WeatherAPI
3. API returns weather data
4. Data is displayed on the UI

---

## 📦 Tech Stack

* HTML
* CSS / Tailwind CSS
* JavaScript
* WeatherAPI

---

## 🔑 API Integration

* The app uses **WeatherAPI** to fetch real-time weather data
* You need an API key to use it

Example API endpoint:

```
https://api.weatherapi.com/v1/current.json?key=YOUR_API_KEY&q=LOCATION
```

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/imabhishekraaz/Weather-app
```

2. Navigate to the project folder:

```
cd weather-app
```

3. Add your WeatherAPI key in your JavaScript file:

```js
const apiKey = "YOUR_API_KEY";
```

4. Run the project:

* Open `index.html` in your browser
